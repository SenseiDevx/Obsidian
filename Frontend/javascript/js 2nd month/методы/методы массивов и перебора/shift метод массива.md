`shift` - это метод массива, который удаляет первый элемент из массива и возвращает удаленный элемент. Этот метод изменяет исходный массив
синтаксис например:
let arr = ['a', 'b', 'c', 'd']  
console.log(arr)
arr.shift('a')  
console.log(arr)
выведет в console:
['a', 'b', 'c', 'd']
[ 'b', 'c', 'd']
