`push` - это метод массива, который позволяет добавить один или более элементов в конец массива. Он возвращает новую длину массива.
синтаксис например:
let arr = ['a', 'b', 'c', 'd']  
console.log(arr)
arr.push('e')  
console.log(arr)
выведет в console:
['a', 'b', 'c', 'd']
['a', 'b', 'c', 'd', 'e']
