`pop` - это метод массива, который удаляет последний элемент из массива и возвращает удаленный элемент. Этот метод изменяет исходный массив.
синтаксис например:
let arr = ['a', 'b', 'c', 'd']  
console.log(arr)
arr.pop('d')  
console.log(arr)
выведет в console:
['a', 'b', 'c', 'd']
['a', 'b', 'c']
