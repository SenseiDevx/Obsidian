`unshift` - это метод массива, который добавляет один или более элементов в начало массива и возвращает новую длину массива. Этот метод изменяет исходный массив.
синтаксис например:
let arr = ['a', 'b', 'c', 'd']  
console.log(arr)
arr.unshift('f')  
console.log(arr)
выведет в console:
['a', 'b', 'c', 'd']
[ 'f',' a', 'b', 'c', 'd']
